# Product Overview

COLABS is an e-commerce platform with multiple components including:

- **Admin Web Interface**: Management dashboard for administrators
- **E-Invoice Web**: Electronic invoicing system for merchants
- **Backend Services**: API and business logic layer via AppSync GraphQL
- **Database Layer**: Multi-stack data storage and analytics

The platform supports multi-environment deployments (dev, staging, prod) with automated CI/CD pipelines for each component.

## Key Features
- Electronic invoicing and merchant management
- Admin dashboard for platform management
- GraphQL API via AWS AppSync
- Multi-tenant architecture with Cognito authentication
- Analytics and search capabilities via Elasticsearch
